API overview
------------

RestrictedPython has tree major scopes:

1. ``compile_restricted`` methods:

  * ``compile_restricted``
  * ``compile_restricted_exec``
  * ``compile_restricted_eval``
  * ``compile_restricted_single``
  * ``compile_restricted_function``

2. restricted builtins

  * ``safe_builtins``
  * ``limited_builtins``
  * ``utility_builtins``

3. helper modules

  * ``PrintCollector``
